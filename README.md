# SevaLink AI 🌍

**AI-powered civic problem reporting and volunteer matchmaking platform.**

## Stack
- **Backend**: Node.js + Express + MongoDB + Google Gemini AI
- **Frontend**: Next.js + Tailwind CSS + Leaflet Maps

## Quick Start

### Backend
```bash
cd backend
npm install
# Add your .env values (MONGO_URI, GEMINI_API_KEY)
npm run dev
```

### Frontend
```bash
cd frontend
npm install
npm run dev
```

Open http://localhost:3000
